# DevMan 前端 UI/UX 设计文档

## 设计原则

1. **开发者优先** — 界面信息密度高，功能直达，减少点击层级
2. **深色主题** — 长时间使用的开发工具，深色减少视觉疲劳
3. **数据驱动** — 用图表和数字说话，直观展示环境状态
4. **即时反馈** — 操作后立即显示结果或进度

## 组件规范

### Panel（卡片容器）
```tsx
// 基础用法
<Panel className="p-6">
  <h3>标题</h3>
  <p>内容</p>
</Panel>

// 样式
bg-devman-panel
border border-devman-border rounded-card
hover:border-devman-border-strong transition-colors
```

### Sidebar（侧边导航）
```
宽度: 240px
背景: devman-bg-deep
选中项: 左侧 accent 色条 + 背景高亮
图标: 24px，文字 14px
```

### 按钮层级
| 类型 | 样式 | 用途 |
|------|------|------|
| Primary | bg-accent text-white | 主要操作 |
| Secondary | bg-panel-raised border | 次要操作 |
| Danger | bg-danger/10 text-danger | 删除/危险 |
| Ghost | hover:bg-panel-raised | 图标按钮 |

### 状态徽章
| 状态 | 颜色 | 文字 |
|------|------|------|
| 健康 | bg-green-500/10 text-green-400 | 🟢 良好 |
| 警告 | bg-yellow-500/10 text-yellow-400 | 🟡 建议清理 |
| 危险 | bg-red-500/10 text-red-400 | 🔴 危险 |
| 信息 | bg-blue-500/10 text-blue-400 | ℹ️ 提示 |

## 响应式设计

当前为桌面应用固定窗口（默认约 1200×800），暂不需要移动端适配。未来支持窗口缩放时：

- **≥ 1200px**: 双栏布局（Sidebar + 主内容）
- **< 1200px**: 窄边栏模式（仅图标）
- **< 900px**: 单栏布局（Sidebar 收起为汉堡菜单）

## 动画规范

| 场景 | 动画 | 时长 |
|------|------|------|
| 页面切换 | fade + slide | 200ms |
| 数据加载 | skeleton pulse | 无限 |
| 按钮 hover | scale(1.02) + shadow | 150ms |
| 进度条 | width transition | 300ms ease-out |
| Toast 弹出 | slide-up + fade | 250ms |
| Modal 遮罩 | fade | 150ms |
